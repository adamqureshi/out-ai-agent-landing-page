# OUT AI Agent landing page (static)

This is a lightweight, static landing page spec for OUT AI Agent (OnlyUsedTesla.ai).

## What’s included
- Light gray / minimal design
- Hero section
- Demo section with 2 selectable product cards:
  - Chat AI Agent (opens chat drawer)
  - Voice AI Agent (opens voice modal — best-effort browser speech APIs)
- Footer

## Notes for your developer
- This is front-end only (no backend).
- Hook up chat + voice to Azure endpoints later.
- Replace links in the nav/footer as needed.

## Run locally
Just open `index.html` in your browser.
